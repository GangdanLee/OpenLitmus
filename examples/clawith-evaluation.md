## 项目概况

Clawith 是 DataElem（数据智元）推出的开源多 Agent 协作平台，定位"OpenClaw for Teams"。核心差异化是 Aware 自主意识系统和数字员工范式。代码量约 42K 行（Python 29K + TypeScript/TSX 12K），技术栈 FastAPI + React 19。项目 Git 公开历史仅 5 天（2026-03-09 至 03-14），此前为私有开发。

### ① 代码规模与结构 — 5.0/10

| 指标 | 发现 | 得分 |

## 总结

**Clawith 是一个有独特产品理念但工程成熟度严重不足的早期项目。**

**值得肯定**: Aware 自主意识系统的设计理念、Trigger Daemon 的工程实现、自研 LLM 客户端的设计决策、中国部署友好的优化，以及清晰的产品差异化定位。

**必须正视**: 零测试 + 零 CI 是一切质量声明的根基性缺失。Git 历史真实性问题使得"成熟迭代"的叙事基础不成立。单人核心贡献者模式下，任何人员变动都可能导致项目停滞。

**最终判断**: 作为技术调研对象有价值（理解多 Agent 协作的产品形态），但距离生产可用有系统性差距。
